# Controls
Move Right -> Right arrow
  
Move Left -> Left Arrow
  
Soft Drop -> Down Arrow

Hard Drop -> Spacebar

Quit -> Q

Restart -> R


# How to run
1. Open Matlab
2. Press Play
3. Follow instructions
